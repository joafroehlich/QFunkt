Wie lautet die Funktionsgleichung einer 
Parabel der Form y = ax2 + bx + c, wenn 
ihr Graph durch die Punkte P1(1|17), 
P2(-4|82) und P3(2|22) geht? 

P1 ergibt:  

x = 1 und y = 17 

Eingesetzt in y = ax2 + bx + c 

17 = a * 12 + b * 1 + c 

a + b + c = 17   (1) 

P2 ergibt:  

x = -4 und y = 82  

Eingesetzt in y = ax2 + bx + c 

82 = a * (-4)2 + b * (-4) + c 

16a – 4b + c = 82   (2) 

P3 ergibt:  

x = 2 und y = 22  

Eingesetzt in y = ax2 + bx + c 

22 = a * 22 + b * 2 + c 

4a + 2b + c = 22  (3)
